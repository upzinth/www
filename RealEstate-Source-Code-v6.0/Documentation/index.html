Admin Panel & API for PHP Laravel

A) Installation And Run Project - top

Admin Panel is a PHP MVC application that developed with Laravel 5.8 and Laravel 8.0. This is a responsive design with bootstrap.

First install this for Laravel 9.0: https://laravel.com/docs/9.x

And set the configurations in .env file; 

DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=publ_realty
DB_USERNAME=root
DB_PASSWORD=Aa123456

MAIL_MAILER=smtp
MAIL_HOST=mailhog
MAIL_PORT=1025
MAIL_USERNAME=null
MAIL_PASSWORD=null
MAIL_ENCRYPTION=null
MAIL_FROM_ADDRESS="hello@example.com"
MAIL_FROM_NAME="${APP_NAME}"

SEEDER_USER_FIRST_NAME=John
SEEDER_USER_LAST_NAME=Publ
SEEDER_USER_USERNAME=admin
SEEDER_USER_PASSWORD=123456

And then run these commands below;

  • 1 - composer update
  • 2 - php artisan migrate
  • 3 - php artisan db:seed
And start the project: php artisan serve

Run with ip address on localhost: php artisan serve --host 192.168._._

B) Authentication - top

For user authentication Laravel Passport is used in mobile, cookies are use in admin panel.

You can check out the controller classes: AccountController and LoginController

C) Controllers - top

The controllers in the project are;

React Native Project

A) Installation And Run Project - top

Real Estate Mobile App is a React Native project. And the project works on Expo tool.

This project works with the following tools;

  • node
  • npm
  • expo cli
Download link for node and npm: https://nodejs.org/en/

For Expo; 
	npm install -g expo-cli

Run project;

Run below command on project's root directory; 

	1. yarn
	2. yarn start

For more information (The link includes information about setup both admin panel and mobile application): https://www.youtube.com/watch?v=Z_-5iHO7P6c

B) App Configurations - top

  • Application Icon
    If you want to change application icon, you can change "icon.png" file in assets.

  • Splash Screen Image
    If you want to change splash screen image, you can change "splash.png" file in assets.

  • Toolbar Logo
    If you want to change toolbar logo image, you can change "logo.png" file in assets.

C) Directory Structure - top



In "src" directory;
  • components
    Contains UI components.
  • models
    Contains service models for screens.
  • navigations
    Contains navigation configurations.
  • localization
    Contains localization files.
  • modals
    Contains bottom sheet modals.
  • screens
    Contains all screens.
  • theme
    Contains theme defines.
  • services
    Dummy services for screens.
